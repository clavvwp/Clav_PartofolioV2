Priority order untuk memulai customize:

1️⃣  EDIT SCRIPT.JS TERLEBIH DAHULU
   👉 Buka: script.js
   👉 Cari: // ===== EDITABLE: PROJECT DATA =====
   👉 Ubah nama project, tahun, kategori, dan nama file gambar
   
2️⃣  TAMBAH GAMBAR PROJECT
   👉 Siapkan 6 file gambar (atau sesuai jumlah project)
   👉 Rename: project1.jpg, project2.jpg, dst...
   👉 Copy ke folder: /img/
   
3️⃣  EDIT INDEX.HTML
   👉 Buka: index.html
   👉 Ubah hero title (cari "Creating next level")
   👉 Ubah subtitle text
   👉 Ubah Contact email (cari "mailto:")
   👉 Ubah Social links
   
4️⃣  EDIT STYLE.CSS (OPSIONAL)
   👉 Buka: style.css
   👉 Ubah warna di :root { ... }
   👉 Terutama: --text-accent (highlight color)
   
5️⃣  TEST DI BROWSER
   👉 Buka index.html dengan double-click
   👉 Cek apakah gambar semua muncul
   👉 Cek responsive dengan F12
   👉 Cek tidak ada error di console

COMMON MISTAKES:
────────────────

❌ Gambar tidak muncul?
   ✅ SOLUSI: Nama file di script.js harus PERSIS SAMA dengan file di /img/
   ✅ Contoh: Jika di script.js ada "img/project1.jpg", 
              maka di /img/ folder harus ada file "project1.jpg"

❌ Teks tidak berubah?
   ✅ SOLUSI: Refresh browser (Ctrl+F5) atau clear cache (Ctrl+Shift+Del)

❌ Animasi tidak jalan?
   ✅ SOLUSI: Buka browser console (F12 → Console tab) dan cek error message

❌ Tombol email tidak bisa?
   ✅ SOLUSI: Pastikan format benar: mailto:your-email@domain.com

❌ Mobile tidak responsive?
   ✅ SOLUSI: Tekan F12 → toggle device toolbar (Ctrl+Shift+M) → resize
